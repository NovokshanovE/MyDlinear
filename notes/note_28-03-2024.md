## Нормализация

Для нормализации данных была применена следующая формула $$t_{norm}=\frac{t-t_{min}}{t_{max}-t_{min}}$$


```python
def minmaxscaler(data):
    min = np.amin(data)
    max = np.amax(data)    
    return (data - min)/(max-min)

```



Ошибка при такой нормализации оказалась очень высокой.
